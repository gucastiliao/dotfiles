# dotfiles

My dotfiles for macOS development environment.

Edit vars file:
```sh
cp vars.example vars
vim vars
```

Execute install:
```sh
sh install.sh
```

- xcode
- brew
    - tmux
    - zsh
    - neovim
    - universal-ctags
    - the_silver_searcher
    - ffmpeg
    - youtube-dl
    - gnupg
    - wget
    - alacritty
    - dbeaver
    - java (8 & latest)
- plug.vim
- oh-my-zsh
- asdf (go, node, python)
